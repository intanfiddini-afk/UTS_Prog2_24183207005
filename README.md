Class Mahasiswa adalah class yang digunakan untuk menampung data setiap mahasiswa. Di dalamnya ada tiga data yang disimpan, yaitu nama, npm, dan jurusan. Class ini memiliki konstruktor untuk mengisi data saat objek dibuat, setter dan getter untuk mengubah dan mengambil nilai data, serta method tampilkanData() untuk menampilkan data mahasiswa ke layar.
